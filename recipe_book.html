Full Stack Recipe Book App

Description:
The Full Stack Recipe Book is a web application designed to help users discover, save, and manage recipes with ease. This project combines a user-friendly interface with powerful backend technology to deliver a seamless experience for cooking enthusiasts. By leveraging APIs, server-side logic, and responsive design, the app ensures that users can access and organize their favorite recipes efficiently on any device.

Technology Stack

For the development of the Full Stack Recipe Book, I used the following technologies:

  • HTML & CSS: For structuring and styling the application, ensuring a clean and user-friendly design.
  • Bootstrap: Employed for its responsive design framework, allowing the app to adapt seamlessly across devices.
  • Vanilla JavaScript: Used to add interactivity and manage dynamic content within the application.
  • Node.js: Utilized for server-side logic, API integration, and handling backend processes.
  • AWS RDS with MySQL: Serves as the database, providing reliable and scalable data storage for recipes.
  • Spoonacular API: Integrated to fetch random recipes, providing dynamic content and enhancing the overall user experience.
  • Version Control: Managed with Git and GitHub, ensuring efficient tracking of changes and collaboration.

My Coding Process

The development of this Full Stack Recipe Book involves a comprehensive approach using modern web technologies. Starting with Bootstrap, I crafted a responsive and aesthetically pleasing user interface. For the backend, Node.js efficiently handles server-side logic, and AWS RDS with MySQL is used for database management. Vanilla JavaScript adds interactivity and dynamic content management on the front end. This stack ensures a robust, scalable, and high-performance application. React may be added in the future for enhanced user experience.

Accessibility in my Coding Process

I am dedicated to ensuring my application is accessible to all users, including those with disabilities. To achieve this, I will conduct testing using the NVDA screen reader and the AXE extension, adhering to WCAG 2.0 standards throughout the development process. As my project testing progresses, I will document my accessibility testing efforts and eventually add pictures to showcase my testing methodologies and results, highlighting my commitment to creating an inclusive application.
