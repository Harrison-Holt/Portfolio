Audio Player App

Description: The Audio Player App is a user-friendly web application that provides controls for playing, pausing, rewinding, and forwarding audio files. It also allows users to dynamically manage a playlist, showcasing skills in frontend development and interactive JavaScript functionality.

Key Features and Planned Improvements:

  • Basic Audio Controls: Features basic audio controls, including play, pause, rewind, and forward, with a display for current playback time.
  • Dynamic Playlist Management: Allows users to add and remove titles from the playlist, demonstrating dynamic DOM manipulation and user interaction capabilities.
  • Enhanced User Experience: Potential improvements include integrating backend services to save playlists, user authentication for personalized settings, and expanding the audio player to support different file types and sources.

Technology Stack

  • Bootstrap: Utilized for its responsive grid system and pre-built components to ensure a seamless look and feel across all devices.
  • HTML & CSS: Used for structuring and styling the application, focusing on a clean and simple design.
  • JavaScript: Implemented for adding interactive elements and dynamic functionalities, enhancing the user experience by making the application more responsive and engaging.
  • Neon DB: Used for managing data and providing backend support to the application, ensuring interactive and stateful behavior.
  • Python: Integrated for backend functionality, contributing to the interactive and dynamic features of the application.
  • Version Control: All code changes were managed with Git and hosted on GitHub to facilitate tracking, versioning, and collaboration.
