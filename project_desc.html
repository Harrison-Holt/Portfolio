Project Descriptions

Personal Portfolio Website

Description:
A showcase of my journey into web development. This project was my first foray into using HTML and CSS. It's structured to highlight my projects, skills, and experiences, demonstrating the foundational practices of web page layout and design.

My Design Process

For my initial portfolio page, I aimed for a clean and minimalist design to show that a simple design can be effective for a webpage.

Wireframe

Color Palette

Color Palette

The color palette I chose for this project reflects the minimalist aesthetic I aimed for. I selected neutral tones with a few accent colors to create a visually appealing yet simple design:

#70BEBC (Soft Seafoam)

#F8F4DE (Warm Ivory)

#E22728 (Vibrant Coral)

My Coding Process

Technologies

For this first project, I focused on improving my HTML/CSS skills and also learned how to use basic Bootstrap to structure the layout of my page.

Usability Testing

To ensure that my portfolio is user-friendly and accessible to a wide audience, I conducted usability tests with six participants from diverse backgrounds and age groups. The feedback from these tests helped identify areas for improvement and refine the overall design.

Participants

  • Engineering Major (22 years old)
  • Architect (25 years old)
  • Software Developer (36 years old)
  • Retired Doctor (68 years old)
  • Anthropology Major (18 years old)
  • Nurse (55 years old)

Testing Methodology

Each participant was asked to complete the following tasks:

  • Find and navigate to a specific project in the portfolio.
  • Describe the design process for one project by locating the provided details.
  • Provide feedback on the overall design, readability, and ease of use.

To-Do List App

Description:
This To-Do List App is a practical demonstration of my foundational skills in web development. As one of my initial projects, it involved utilizing HTML, CSS, and vanilla JavaScript, supplemented by Bootstrap for responsive design. The app allows users to add, manage, and prioritize tasks effectively, showcasing my ability to build functional, user-friendly web interfaces.

My Design Process

I aimed for a clean and minimalist design to ensure that the functionality of the to-do list is straightforward and accessible. This approach highlights the practical application of user-centric design principles in creating effective web solutions.

Wireframe

Color Palette

The color palette was carefully chosen to reflect a minimalist aesthetic while providing visual cues for user interaction:

#FFFFF0 (Ivory White)

#caa478 (Soft Bronze)

#A2B59F (Muted Sage)

#789EC5 (Calm Sky Blue)

My Coding Process

Technologies

This project allowed me to deepen my understanding of HTML and CSS, and to explore basic JavaScript functionalities. I integrated Bootstrap to take advantage of its built-in responsive design features, making the to-do list accessible across different devices and screen sizes.
