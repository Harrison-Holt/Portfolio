NBA Card Collection App

Description: The NBA Player Card Collection App is designed to showcase data management and interactive features by allowing users to collect and manage cards of NBA players. This project highlights the use of backend services to handle data efficiently and demonstrates the ability to create engaging user experiences around data visualization and manipulation.

Key Features and Planned Improvements:

  • Data Handling: Efficiently manages large datasets of player information, showcasing backend skills in handling and serving data.
  • Interactive Features: Includes interactive elements such as searching, filtering, and sorting cards based on player statistics and other attributes.
  • Customization: Future enhancements could include real-time updates, user authentication, personalized collections, and social sharing features to make the app more engaging

Technology Stack

  • Bootstrap: Utilized for its responsive grid system and pre-built components to ensure a seamless look and feel across all devices.
  • HTML & CSS: Used for structuring and styling the application, focusing on a clean and simple design.
  • JavaScript: JavaScript is employed to introduce dynamic and interactive features to the application, such as real-time updates, animations, and form validations. It plays a crucial role in making the user interface more responsive and engaging by enabling actions like searching, filtering, and sorting through the NBA player cards.
  • Neon DB: Used for managing data and providing backend support to the application, ensuring interactive and stateful behavior.
  • Golang: Golang (Go) is integrated for backend functionality, responsible for handling server-side logic, managing API requests, and interacting with the database. It powers the core functionalities of the application, such as user authentication, data processing, and real-time updates.
  • Version Control: All code changes were managed with Git and hosted on GitHub to facilitate tracking, versioning, and collaboration.
