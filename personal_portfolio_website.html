Personal Portfolio Website

Description:
A showcase of my journey into web development. This project was my first foray into using HTML and CSS. It's structured to highlight my projects, skills, and experiences, demonstrating the foundational practices of web page layout and design.

My Design Process

For my initial portfolio page, I aimed for a clean and minimalist design to show that a simple design can be effective for a webpage. I selected a color palette that reflects this aesthetic, using neutral tones with a few accent colors to create a visually appealing yet simple design:

#70BEBC (Soft Seafoam)

#F8F4DE (Warm Ivory)

#E22728 (Vibrant Coral)

I used Merriweather and Cardo as fonts to maintain readability and a clean look. One of the challenges I faced was struggling with Bootstrap initially, but I overcame this by practicing and gradually getting comfortable with its components and grid system.

Wireframe and Design Mockup

I used Figma to create the wireframe and design mockup for this project. Initially, I planned for a single-page design as seen in my wireframe. However, I realized that the page was becoming too long and cluttered, which led me to add an extra page in my design mockup to improve usability and navigation.

My Coding Process

Technologies

For this first project, I focused on improving my HTML/CSS skills and also learned how to use basic Bootstrap to structure the layout of my page. This project was all about making clean HTML and using semantic code. I learned Bootstrap and how to utilize it for layout with flexboxes, and used CSS transitions and animations. I wanted the design to be simple, not overpowering, and easy to read.
