To-Do List App

Description:
This To-Do List App is a practical demonstration of my foundational skills in web development. As one of my initial projects, it involved utilizing HTML, CSS, and vanilla JavaScript, supplemented by Bootstrap for responsive design. The app allows users to add, manage, and prioritize tasks effectively, showcasing my ability to build functional, user-friendly web interfaces.

My Design Process

I aimed for a clean and minimalist design to ensure that the functionality of the to-do list is straightforward and accessible. This approach highlights the practical application of user-centric design principles in creating effective web solutions. I used Figma to create the wireframe and design mockup for this project.

Technology Stack

For the development of the To-Do List App, I used the following technologies:

  • HTML & CSS: Used for structuring and styling the app, with a focus on clean and semantic code.
  • Bootstrap: Employed for its responsive grid system and components, making the app adaptable to various screen sizes.
  • JavaScript: Leveraged for dynamic functionality, such as task management and date handling, without the use of frameworks.
  • Figma: Utilized for wireframing and creating the design mockup, guiding the visual and structural layout of the app.
  • Ninja Quotes API: Used to fetch motivational quotes dynamically, adding an inspirational element to the user's daily tasks.
  • Version Control: Managed with Git and GitHub, ensuring efficient tracking of changes and collaboration.

My Coding Process

This project allowed me to deepen my understanding of HTML and CSS, and to explore basic JavaScript functionalities. I integrated Bootstrap to take advantage of its built-in responsive design features, making the to-do list accessible across different devices and screen sizes. I wanted to learn JavaScript and all its functionality without the need for frameworks. My design was to assemble the app to look like a bulletin board where you thumbtack post-it notes to the board. The biggest challenge I faced was learning how to create HTML elements dynamically and working with the Date class. Additionally, I added a "Quote of the Day" feature to provide daily motivation to the user. I used CSS with a little Bootstrap and vanilla JavaScript only.

Accessibility in my Coding Process

I am dedicated to ensuring my application is accessible to all users, including those with disabilities. To achieve this, I will conduct testing using the NVDA screen reader and the AXE extension, adhering to WCAG 2.0 standards throughout the development process. As my project testing progresses, I will document my accessibility testing efforts and eventually add pictures to showcase my testing methodologies and results, highlighting my commitment to creating an inclusive application.
