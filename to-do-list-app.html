To-Do List App

Description:
This To-Do List App is a practical demonstration of my foundational skills in web development. As one of my initial projects, it involved utilizing HTML, CSS, and vanilla JavaScript, supplemented by Bootstrap for responsive design. The app allows users to add, manage, and prioritize tasks effectively, showcasing my ability to build functional, user-friendly web interfaces.

My Design Process

I aimed for a clean and minimalist design to ensure that the functionality of the to-do list is straightforward and accessible. This approach highlights the practical application of user-centric design principles in creating effective web solutions. I used Figma to create the wireframe and design mockup for this project. Initially, I wanted a single-page design, but I realized it was too long and cluttered, so I added extra sections to improve usability.

My Coding Process

Technologies

This project allowed me to deepen my understanding of HTML and CSS, and to explore basic JavaScript functionalities. I integrated Bootstrap to take advantage of its built-in responsive design features, making the to-do list accessible across different devices and screen sizes. I wanted to learn JavaScript and all its functionality without the need for frameworks. My design was to assemble the app to look like a bulletin board where you thumbtack post-it notes to the board. The biggest challenge I faced was learning how to create HTML elements dynamically and working with the Date class. Additionally, I added a "Quote of the Day" feature to provide daily motivation to the user. I used CSS with a little Bootstrap and vanilla JavaScript only.
